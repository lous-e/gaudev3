# BidMesh Agent View

Status: concept surface only
Mode: machine-readable negotiation runtime

## Promise

Agents can choose tactics.
Agents cannot expand authority.

BidMesh turns negotiation into a deterministic loop:
- search sellers
- compare viable quotes
- counter within budget
- stop at the human approval boundary

## Runtime Model

intent: USB-C cable
currency: USDC
max_price: 5.00
max_rounds: 3
require_human_confirmation_before_payment: true

## Session Preview

tessa.agent@bidmesh:~/sessions/deal_000001
$ negotiate --item "USB-C cable" --cap 5.00 --currency USDC --rounds 3
